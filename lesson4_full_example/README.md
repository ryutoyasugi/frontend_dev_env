# Sass + TypeScript + Gulp

```sh
npm install
npm start # develop
npm run build # build
```

npm modules

* gulp-typescript
* gulp-uglify (minify)
* gulp-sass
* gulp-sass-lint
* gulp-plumber (prevent to stop task when build failed)
* gulp-notify (notification to desktop when build failed)
* gulp-apimocker
* browser-sync
